# MedGuide AI - RAG Powered Health Chatbot

MedGuide AI is an intelligent healthcare chatbot developed using Retrieval-Augmented Generation (RAG) architecture. The system retrieves relevant medical information using cosine similarity and generates accurate, context-aware responses using Claude Sonnet AI. The project is designed to assist users with healthcare-related queries through an interactive and efficient AI-powered interface.

This chatbot improves the quality of responses by combining information retrieval techniques with large language models. Instead of generating responses only from pretrained knowledge, the system first searches for relevant medical context and then produces more reliable and meaningful answers.

The application provides a user-friendly interface where users can ask healthcare-related questions and receive AI-generated responses instantly. The frontend is developed using React.js, while the backend logic integrates RAG architecture, cosine similarity retrieval, and Claude Sonnet API.

## System Workflow

User Query → Embedding Generation → Similarity Search → Relevant Medical Context Retrieval → Claude Sonnet Processing → AI Response Generation

How the System Works

The user enters a healthcare-related query into the chatbot interface. The system converts the query into embeddings and performs cosine similarity search on the stored medical knowledge base. Relevant medical context is retrieved and provided to the Claude Sonnet model. Using this retrieved information, the AI generates an accurate and context-aware response for the user.
